# wenai Operating System

wenai is not positioned as an AI copywriting tool.

It is an AI commercial delivery system for ecommerce teams:

- SKU intake
- category rules
- brand redlines
- content marketing benchmark
- POC report
- contract motion

All in one operating line.

## Core modules

### 1. Launch Pack
- SKU intake
- image direction
- listing copy
- compliance notes
- customer-service FAQ

### 2. Brand IQ
- brand voice rules
- forbidden claims
- category acceptance thresholds
- review-owner requirement
- default workflow rules

### 3. Growth Test Pack
- TikTok / Instagram benchmark
- hook matrix
- slideshow brief
- reel brief
- 7-day publishing report

### 4. POC Report
- acceptance score
- benchmark coverage
- missing asset count
- rework count
- contract intent signal

### 5. CRM Pipeline
- inquiry state
- contract stage
- quote status
- payment status
- SLA due date
- next action

## Product moat

wenai moat does not come from one model output.

It comes from:

1. Ecommerce category guardrails
2. Brand IQ and forbidden-claim memory
3. SOP-standardized deliverables
4. POC scoring and review logic
5. CRM contract-motion structure
6. Evidence-backed content workflow

## Sellable promise

Give us 10 real SKUs and the target platform.

wenai turns them into:

- a launch-ready standard pack
- a reviewable POC report
- a benchmark-based content plan
- a contract-ready operating record

## What wenai does not promise

- one-click final publishing without human review
- legal or trademark final approval
- fake testimonials or invented growth proof
- replacing the customer owner in final sign-off
