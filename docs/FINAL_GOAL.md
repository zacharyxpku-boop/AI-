# wenai 最终目标

更新时间: 2026-05-02

## 一句话目标

让 `wenai` 成为一个能稳定获客、稳定交付、稳定复盘、稳定转合同的 **10 SKU POC 子站**。

它不是一个泛 AI 工具站，也不是一个低价自助 SaaS。它的职责只有一个: 把高意向客户从演示带到真实 POC，再把合格 POC 推进到主站合同和支付。

参考 PostPlus 后，POC 定义升级为:

**10 SKU 上新物料包 + 内容拆解与增长测试包 + 创意生产包 + 复盘指标**

也就是不只生成 listing，还要基于真实市场内容证据，告诉客户这批 SKU 上线前后该参考什么内容结构、怎么改编、用什么素材测试、怎样做出内容成品、看什么数据、怎样复盘是否继续投入。

## 北极星指标

**每月进合同的合格 POC 数**

解释:
1. 不是总询盘数。低质量询盘越多，团队越忙，商业化越差。
2. 不是总交付数。只交付不进合同，说明产品价值没有闭环。
3. 不是网站流量。这个子站的任务是承接和推进，不是做大而空的内容站。

## 90 天阶段目标

### 第 1 阶段: 线索合格化

目标:
- 询盘表单不再收一堆模糊需求
- 明确 10 个真实 SKU、平台、素材状态、验收口径
- 把低质量线索挡在交付前

硬指标:
- 合格率 >= 35%
- `SKU 不足 / 平台未填 / 验收口径缺失` 三类问题持续下降
- 逾期跟进数保持在低位

### 第 2 阶段: POC 交付标准化

目标:
- 交付不靠临场发挥
- 每个 POC 都能形成验收包、复盘记录、返工边界
- 后台可以直接看出卡点在资料、产能还是客户配合

硬指标:
- 合格线索交付率 >= 60%
- 平均验收分 >= 80
- 已交付项目都能落下复盘记录

### 第 3 阶段: 从 POC 到合同

目标:
- 每个交付都能推到复盘
- 每次复盘都明确下一步: 返工、扩 SKU、或进主站合同
- 把“看起来不错”变成“愿意继续买”

硬指标:
- 复盘率 >= 70%
- 总询盘合同率 >= 15%
- 合格 POC 的合同转化开始稳定出现

## 我们不做什么

1. 不做“AI OS”叙事，不堆 100 个泛功能页面。
2. 不做低价自助工具，不跟平台原生 AI 功能打价格战。
3. 不在这个仓库里做正式支付链路，这里只负责演示、承接、推进。
4. 不做假案例、假 ROI、假客户背书。
5. 不做高人力定制代理服务伪装成产品。

## 真正护城河

不是模型本身，而是下面这些可复用资产:

1. **线索结构化能力**  
   知道哪些客户能做，哪些客户不该做。

2. **10 SKU POC 验收包模板**  
   输出形态统一，方便复盘、对比、签单。

3. **交付过程数据**  
   哪类客户更容易通过验收，哪类素材最容易卡住，哪个来源更容易进合同。

4. **类目化 prompt 和人工终审边界**  
   不是泛生成，而是能落在真实电商上新流程里。

5. **子站到主站的商业承接路径**  
   这个子站只做最擅长的一段，不承担全部商业复杂度。

6. **Benchmark-to-campaign 数据**  
   把 SKU、参考内容、用户评论、脚本结构、素材清单、平台反馈和复盘结果连起来，而不是只交一份静态文案。

7. **Creative Production Pack 模板**  
   把 Podcast UGC、Street Interview、Batch UGC、Slideshow、Animated Ads、Editing 这些高价值创意工作流产品化，而不是每次临时重做。

## Kill Criteria

如果连续 90 天出现以下情况，就不要继续给自己找借口:

1. 没有出现稳定的合格询盘来源
2. POC 交付后没有客户愿意进入复盘
3. 复盘后依然没有任何合同推进信号

这说明问题不在功能不够多，而在定位、样例、入口或目标客户本身。

## 当前执行顺序

1. 继续压缩入口和询盘文案，让线索更清晰
2. 继续强化后台项目板和指标板，让每个 POC 有跟进动作
3. 继续固化验收包和复盘模板，减少交付波动
4. 把内容拆解与增长测试包纳入 POC，学习 PostPlus 的 evidence-first 流程但保持电商 SKU 主线
5. 把创意生产包纳入高阶交付层，只做能成交的工作流模板
6. 继续追踪来源质量、验收分和合同推进，形成学习闭环

## 这个仓库的判断标准

以后评估 `wenai`，不要再问“功能多不多”，而要问:

1. 有没有带来更高质量的 POC 线索
2. 有没有让交付更稳定
3. 有没有让复盘更顺
4. 有没有让合同推进更强

满足这四条，才算在往最终目标走。
