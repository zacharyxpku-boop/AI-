# wenai Demo Room

Use these samples when showing the product before real customer data is connected.

## Sample 1: Home Decor

- Category: Home decor
- SKU: foldable kitchen storage rack
- Platform: Amazon / Shopify
- Brand redlines: avoid medical, safety, or unrealistic durability claims
- Expected pack: listing copy, image direction, compliance reminders, support FAQ, executive recap

## Sample 2: Auto Parts

- Category: Auto parts
- SKU: magnetic dashboard phone mount
- Platform: Amazon / TikTok Shop
- Brand redlines: compatibility must be reviewed manually
- Expected pack: install context, compatibility warning, claims review, short-video hook ideas

## Sample 3: Electronics

- Category: Electronics
- SKU: wireless earbuds with low-latency mode
- Platform: Shopify / Instagram
- Brand redlines: avoid Apple, AirPods, Sony, or unsupported certification references
- Expected pack: benefit angles, spec normalization, trademark reminders, content testing brief
