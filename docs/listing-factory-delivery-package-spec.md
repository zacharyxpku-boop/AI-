# Wenai Listing Factory 交付包规格

## 交付包包含什么

- POC 报告
- Brief CSV
- 脚本 Markdown
- 分镜 Markdown
- 素材清单 Markdown
- 缺失素材 Checklist
- Assembly Manifest Markdown / CSV
- 批量生产批次 Markdown
- Edit Pack Markdown
- 字幕 SRT 示例
- EDL CSV
- Asset Manifest CSV
- QA Summary
- 客户消息草稿

## 每个文件给谁看

- POC 报告：客户、老板、商务负责人
- Brief CSV：内容运营、顾问、后续复盘
- 脚本 / 分镜：剪辑师、AI 视频工具、内容运营
- 素材清单 / 缺失素材 Checklist：拍摄/素材同事、运营、顾问
- Assembly Manifest：剪辑师、制作同事
- 批量生产批次 / Edit Pack：剪辑师、客户交付负责人
- SRT / EDL / Asset Manifest：剪辑执行、内部质检
- QA Summary：运营、顾问、制作负责人
- 客户消息草稿：客户经理、顾问

## 怎么使用

1. 先看 POC 报告判断要不要推进首轮生产。
2. 再看 Batch / Edit Pack 了解每条内容怎么剪、怎么配素材。
3. 用 SRT、EDL、Asset Manifest 进入剪辑或 AI 视频工具。
4. 用 QA Summary 判断是否需要先补素材或复核风险。

## 正式交付 / 内部工作文件

- 正式交付：POC 报告、客户消息草稿、Brief CSV、Edit Pack、SRT、EDL、Asset Manifest
- 内部工作文件：素材清单、缺失素材 Checklist、Assembly Manifest、QA Summary、批量生产批次

## local-first 边界

- 数据只保存在当前浏览器
- 可导出 JSON 延续项目
- 不接真实云盘、不接真实视频生成、不接平台 API

## 未来接真实 LLM / 平台 API 时哪些不变

- 交付包结构不变
- 文件分工不变
- QA 检查项不变
- 客户阅读方式不变
