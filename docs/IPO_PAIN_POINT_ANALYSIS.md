- 产品开拓：市场调研找潜力品、成本风控、供应商开发维护
- 产品采购：采购下单、回货跟踪、货款申请、系统录入
- 产品生产：服装生产相关
- 物流仓储：最多人数

**结论：跨境电商企业80-85%的人力被供应链和营销吃掉，这是行业铁律。**

## 二、两家公司共同风险因素（=行业痛点）

### P0级痛点（所有跨境电商都逃不掉）

1. **平台依赖风险**
   - 尚睿：亚马逊B2C收入占比 66-74%
   - 赛维：亚马逊收入占比 70-90%（逐年上升）
   - 平台政策变动、封号、费率调整 = 生死线

2. **存货管理风险**
   - 尚睿：存货占资产28-44%，逐年上升
   - 赛维：存货6-8.9亿，周转率2.25-2.65次
   - 选品失误 = 库存积压 = 资金链断裂

3. **海运价格波动**
   - 2021-2022年海运暴涨直接吃利润
   - 国际物流成本不可控

4. **汇率波动**
   - 尚睿：汇兑收益占利润7-14%
   - 美元/欧元结算，汇率波动直接影响利润

5. **国际贸易政策**
   - 尚睿：美国销售占比80-85%
   - 关税、合规要求随时可能变化

### P1级痛点（影响增长但不致命）

6. **季节性波动**
   - 下半年（Q4）旺季，上半年淡季
   - 需要精准的备货+营销节奏

7. **多账号经营合规风险**（赛维特有但行业通用）
   - 亚马逊多账号模式存在被封风险
   - 平台政策解释权在平台手里

8. **品牌影响力维护**
   - 赛维63个营收过千万品牌，管理复杂
   - 品牌矩阵 vs 品牌稀释

9. **人才流失**
   - 跨境电商人才竞争激烈
   - 高素质运营/采购人才稀缺

### P2级痛点（隐性但重要）

10. **境外合规（知识产权+隐私+消费者保护+反不正当竞争）**
    - 尚睿明确提到需遵守不同司法管辖区的知识产权、隐私保护、消费者权益保护
    - 合规失误 = 经济损失

11. **IT系统跟不上变化**（赛维提到）
    - 跨境电商行业变革快，系统需持续迭代
    - 全链路数字化（产品开发→供应链→销售运营）是竞争力

## 三、对Wenai产品的启示

### 哪些模块对准了真实痛点？

| Wenai模块 | 对应痛点 | 痛点等级 | 能否AI化 |
|-----------|----------|----------|----------|
| 批量翻译 | 多语言多市场运营 | P0 | ✅ 高度可AI化 |
| 评论采集分析 | 选品/存货决策辅助 | P0 | ✅ 高度可AI化 |
| IP侵权防控 | 境外合规风险 | P2→P0（一旦出事就致命） | ✅ 规则+数据库可AI化 |
| 商品文案 | 营销人员效率 | P1 | ✅ 高度可AI化 |
| 媒介外联 | 营销人员效率 | P1 | ✅ 模板+个性化 |
| AI客服 | 售后服务成本 | P1 | ✅ FAQ+自动回复 |
| 竞品拆解 | 选品/运营决策 | P1 | ⚠️ 需要实时数据 |
| 选品辅助 | 存货管理核心 | P0 | ⚠️ 只能辅助，最终靠人 |
| 运营策略 | 营销节奏/投放 | P1 | ⚠️ 需行业数据 |

### 招股书验证的关键认知

1. **选品确实只能辅助**
   - 赛维的供应链人员要做：市场调研→成本风控→供应商开发→采购下单→回货跟踪
   - 用户说的对："很多都是跟工厂沟通定制模具，产品颗粒度抓的很严"
   - AI能帮数据筛选，但供应商谈判、模具定制、质量验收必须人做

2. **真正的人力黑洞在"营销+运营"（占比40-50%）**
   - 这才是AI最大的切入点
   - 文案/翻译/评论分析/客服 = 营销人员日常消耗最大的重复劳动

3. **IP合规是"低频高致命"痛点**
   - 招股书明确列为风险因素
   - 一次侵权 = 店铺封禁 = 直接损失百万级
   - 值得做，但需要真实数据库支撑

4. **存货管理是"老板级"痛点**
   - 两家公司都重点提到
   - 但这个需要对接ERP/库存系统，不是纯AI能解的

### 优先级调整建议

**第一梯队（立即能用，锚点客户最关心）：**
1. 批量翻译 — 5品类×N市场，刚需
2. 评论采集分析 — 辅助选品+优化listing
3. 商品文案/Listing优化 — 营销人员日常高频
4. IP侵权防控 — 防御型，出事就是大事

**第二梯队（需要数据支撑）：**
5. 竞品拆解 — 需对接蝉妈妈等数据源
6. 选品辅助 — 数据筛选层面
7. AI客服 — 需要品类知识库

**第三梯队（锦上添花）：**
8. 媒介外联
9. 运营策略
10. 种草内容/直播脚本
