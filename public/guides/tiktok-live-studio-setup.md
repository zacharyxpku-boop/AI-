## 2. 直播间环境选择

### 空间要求（按优先级）

1. **独立、安静** — 能关门，无家人/同事穿梭
2. **面积 ≥ 6 ㎡** — 放得下主播 + 产品陈列 + 补光灯
3. **有平整背景墙** — 白墙/浅灰墙最佳
4. **网络稳定** — 路由器 WiFi 5GHz 频段，或直接有线网口
5. **插座充足** — 至少 4 个插座位

### 背景墙规范

| ❌ 错误做法 | ✅ 正确做法 |
|------------|------------|
| 纯白大白墙 | 浅灰/米白/浅木色 |
| 杂物堆满的家庭背景 | 统一色调的干净背景 |
| 强反光玻璃/镜面 | 哑光背景布/墙纸 |
| 背后有活动人员穿行 | 背后封闭或有遮挡 |

**预算 $30 搞定背景：** 淘宝/Amazon 买 2×3 米的哑光背景布 + 便携支架，灰色/驼色/米白各买一套。

## 3. 三点布光法（新手必学）

一套完整的直播灯光 = 主光 + 辅光 + 背景光，总预算 $150-250。

```
         [背景光]
            |
   [辅光] --主播-- [主光]
            |
         [镜头]
```

### 主光（Key Light）
- **位置：** 主播正前方 45° 斜上方
- **作用：** 主光源，照亮脸部
- **推荐：** Neewer 18 寸环形灯 或 软光板 60×90cm
- **预算：** $50-80

### 辅光（Fill Light）
- **位置：** 主播侧面 45°，主光相反方向
- **作用：** 消除主光造成的阴影
- **亮度：** 约主光的 50-70%
- **推荐：** 小型 LED 板 或 另一个小环形灯
- **预算：** $30-50

### 背景光（Rim/Back Light）
- **位置：** 主播后方，从侧后方打
- **作用：** 把主播从背景中"抠"出来，增加立体感
- **推荐：** 任何带色温调节的 LED 条灯
- **预算：** $30-50

### 布光铁律

| 错误 | 后果 | 修正 |
|------|------|------|
| 头顶正上方打光 | 眼窝黑圈、鼻影重 | 灯往前挪 45° |
| 背后开窗当光源 | 主播变剪影 | 闭窗/拉帘，改前面打光 |
| 所有灯色温不一致 | 肤色发黄或发青 | 全部调成 5600K（日光色） |
| 补光过曝 | 脸部死白、肤色失真 | 调低亮度到 70% |

## 5. 开播前检查清单

> 打印出来贴墙上。每场直播前过一遍。

### T-30 分钟
- [ ] 充电：手机电量 100%，充电宝满电，备用机充满
- [ ] 测试：运行一次直播彩排，看画面/声音/网速
- [ ] 话筒：贴紧、无杂音、左右声道正常
- [ ] 灯光：全部打开，色温一致，无闪烁
- [ ] 网速：Speedtest 上传速度 ≥ 10 Mbps
- [ ] 产品：按讲解顺序摆放，价格牌贴好
- [ ] 脚本：开场 30 秒话术打印放镜头旁
- [ ] 手势：确认所有优惠券/抽奖在后台已设置

### T-5 分钟
- [ ] 清水/温茶放在手边（讲 2 小时必须补水）
- [ ] 手机静音，通知全关
- [ ] 副播就位，运营中控开机
- [ ] 最后一次检查镜头取景框，调整姿态

### 开播
- [ ] 点开播，开始执行"开场 30 秒留人话术"
- [ ] 第 1 分钟内挂上引流福利品
- [ ] 眼神看镜头，不要看自己的屏幕

## 6. 常见问题排查

| 症状 | 可能原因 | 修正动作 |
|------|---------|---------|
| 画面卡顿 | 网速不够/手机发热 | 换有线网/降低清晰度/给手机散热 |
| 声音有回音 | 麦克风距离不对 | 领夹麦贴胸口 15cm 以内 |
| 脸发黄 | 光源色温太暖 | 全部调 5400-5600K |
| 脸死白过曝 | 主光太近 | 主光往后挪 30cm |
| 直播中掉线 | WiFi 不稳 | 换 5GHz 频段/拉网线 |
| 观看人数涨不起来 | 标题/封面没抓住 | 去看同品类 Top 10 直播间的标题规律 |
